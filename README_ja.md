## 概要

MedAISure ベンチマークは、医療分野における大規模言語モデル（LLM）の評価のための包括的なフレームワークです。臨床的な質問応答、診断推論、要約、患者コミュニケーションなど、現実の医療タスクを解決する能力を評価します。
