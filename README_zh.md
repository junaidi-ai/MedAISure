## 概述

MEDDSAI基准测试是一个用于评估医疗领域大语言模型（LLM）的综合框架。它挑战模型解决现实世界中的医疗任务，如临床问答、诊断推理、总结和患者沟通等，这些任务都经过医学专家的验证。
